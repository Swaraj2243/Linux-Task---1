# Part F - Linux Research Activity

## What is Linux?

Linux is a free and open-source operating system used on servers, desktops, cloud platforms, and embedded systems. It is known for its stability, security, and flexibility.

## Why is Linux important in Cyber Security?

Linux is important in cybersecurity because most security tools, servers, and penetration testing environments run on Linux. It provides powerful command-line tools and better system control.

## Difference between Linux and Windows

Linux is open-source, customizable, and generally more secure. Windows is proprietary software developed by Microsoft and is more commonly used on personal computers.

## What is a Linux Distribution?

A Linux Distribution is a version of Linux bundled with software, package managers, and desktop environments. Examples include Kali Linux, Ubuntu, and Parrot OS.

## Why do Ethical Hackers prefer Linux-based operating systems?

Ethical hackers prefer Linux because it offers powerful security tools, customization, stability, scripting capabilities, and better control over the operating system.
