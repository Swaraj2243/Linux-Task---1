# Linux Task 01 - Linux Environment Setup & Essential Commands

## Objective

The objective of this task is to understand Linux basics, terminal usage, navigation, file management, and system information commands.

---

## Part B - Basic Navigation Commands

### pwd

Displays the current working directory.

### ls

Lists files and directories.

### ls -la

Lists all files including hidden files with detailed information.

### cd

Changes the current directory.

### clear

Clears the terminal screen.

### history

Shows previously executed commands.

### whoami

Displays the current logged-in username.

### hostname

Displays the system hostname.

---

## Part D - File Management Commands

### touch

Creates a new empty file.

### cp

Copies files and directories.

### mv

Moves or renames files and directories.

### rm

Deletes files and directories.

---

## Part E - System Information

### Kernel Version

Obtained using uname -a

### Username

Obtained using whoami

### Current Directory

Obtained using pwd

### Current Date and Time

Obtained using date

### System Uptime

Obtained using uptime

This task helped in understanding Linux terminal basics, file management, directory navigation, and system information collection.
